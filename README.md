Description
this is a simple store inventory system that holds items and electronics. it uses python objects to store the data and let you search for an item easily.

How to run
open command prompt or terminal
type python main.py
then the program will ask you to enter an item to search like tv or laptop
then it will ask you to enter a number to update the price

Features
- stores items in a list array
- uses linear search to find an item fast
- has try except so the program doesnt crash if you type a letter instead of a number
- uses inheritance because electronic extends item and uses super
- uses encapsulation by making variables private and using getters and setters to change them securely
- uses polymorphism by overriding the show method so electronics prints different then a regular item
